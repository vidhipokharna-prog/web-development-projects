# Bookstore Website

A static multi-page bookstore website built using HTML and CSS.  
This project focuses on designing a user-friendly interface for an online bookstore.

## Features
- Home page with bookstore overview
- Book categories page
- Fiction and non-fiction book sections
- Shopping cart page
- Login page
- Subscription and purchase pages

## Technologies Used
- HTML
- CSS

## Purpose
This project was created as part of my web development practice to improve
frontend design skills and understanding of website structure.
