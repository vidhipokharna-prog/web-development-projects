# Blood Bank Management System

A web-based blood bank management system developed using HTML, CSS, JavaScript, and PHP.
This project helps manage blood donors, blood availability, and blood requests efficiently.

## Features
- Donor registration system
- Blood group and blood stock management
- Real-time blood availability tracking
- Blood request and approval management
- Dashboard showing:
  - Blood donated
  - Blood received
  - Pending blood requests
- Nearby hospitals listing to check blood availability
- QR-based registration system to reduce paperwork and manual data entry
- Simple and user-friendly admin/user interface

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL (database for storing donor and blood data)

## Purpose
This project was developed as part of academic coursework and backend development practice.
It focuses on understanding server-side programming, database connectivity, and real-world
healthcare system workflows.
